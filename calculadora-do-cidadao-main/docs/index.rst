Calculadora do Cidadão
======================

A `Calculadora do Cidadão <https://github.com/cuducos/calculadora-do-cidadao>`_ requer Python 3.7 ou 3.8. Como ela está `disponível no PyPI <https://pypi.org/project/calculadora-do-cidadao/>`_, pode ser instalada com o `pip`:

::

    $ pip install calculadora-do-cidadao
